## Принципы

1. **`FULFILLED` только после «Беру!»** — random не меняет статус сам.
2. **Pick до анимации** — колесо честное, winner известен заранее.
3. **Цены в minor units** — `Long`, без `Double` в БД.
4. **Один ViewModel**, `safeDb`, ошибки в Snackbar.
5. **`minSdk 26`** — Android 8.0+.

## Риски и митигация

| Риск | Митигация |
|------|-----------|
| Анимация лагает на старых API 26 | Упростить колесо; fallback — fade-in карточки |
| Ошибки ввода цены | Маска + unit-тесты парсера |
| Пустой вишлист | Блокировать «Крутить», CTA в вишлист |
| Путаница «беру» vs реальная покупка | Текст в UI: «отметить как исполненное в списке» |
