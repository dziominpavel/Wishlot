# Proposal: UI Cleanup — убрать стрелки, чекбокс анимации, символ ₽

## Motivation

Текущий UI содержит избыточные элементы и непоследовательное форматирование:
- Стрелки вверх/вниз на каждой карточке желания занимают место и усложняют интерфейс. Пользователь может менять порядок через drag-and-drop или не менять вовсе — ручные кнопки не нужны.
- Переключатель анимации колеса в настройках избыточен: анимация должна быть включена всегда как часть UX.
- Формат цены через `NumberFormat.getCurrencyInstance()` выводит "руб." или "RUB" в зависимости от locale вместо единого символа `₽`.
- Launcher icon требует обновления (вне scope текстовых изменений — отдельно).

## Scope

### In Scope
- Удалить стрелки сортировки (`KeyboardArrowUp`/`KeyboardArrowDown`) из `WishCard` и связанную логику `moveWishUp`/`moveWishDown`.
- Удалить настройку "Анимация колеса" из `SettingsScreen`, `ViewModel`, `SettingsRepository`.
- Заменить форматирование цены в `MoneyUtils.formatMinor()` на `₽` с разделителем тысяч.
- Удалить неиспользуемые строковые ресурсы.

### Out of Scope
- Обновление launcher icon PNG (требует графического ресурса).
- Изменение логики сортировки (сам порядок остаётся).

## Acceptance Criteria
- [ ] В списке активных желаний нет стрелок вверх/вниз.
- [ ] В настройках нет переключателя анимации колеса.
- [ ] Все цены отображаются с символом `₽` (например, `1 250 ₽`).
- [ ] Clean build проходит, приложение не крашится.
