# Delta Spec: Wheel / Удаление приоритета

## Изменения относительно `openspec/specs/wheel/spec.md`

## REMOVED
```markdown
### Requirement: Взвешенный выбор по приоритету
Рулетка SHALL учитывать приоритет желания при выборе победителя.

#### Scenario: Высокий приоритет чаще выигрывает
- GIVEN два желания с разным приоритетом
- WHEN запускается множество розыгрышей
- THEN высокоприоритетное желание выпадает чаще
```

## MODIFIED
- `Requirement: Random не меняет статус` — остаётся без изменений.
- `Requirement: Фильтр по бюджету` — остаётся без изменений.

## ADDED
```markdown
### Requirement: Равновероятный выбор
Все кандидаты на розыгрыш SHALL иметь равную вероятность быть выбранным.

#### Scenario: Равные шансы
- GIVEN три активных желания в рамках бюджета
- WHEN запускается рулетка
- THEN каждое желание имеет одинаковую вероятность 1/3
```
