# Delta Spec: Wishlist / Удаление приоритета

## Изменения относительно `openspec/specs/wishlist/spec.md`

## REMOVED
- Упоминания поля `priority` как атрибута желания.
- Упоминания приоритета как фактора, влияющего на сортировку или поведение.

## MODIFIED
- `Requirement: Создание желания` — убрано требование выбора приоритета. Пользователь заполняет только название, описание, бюджет, ссылку и категорию.

## ADDED
```markdown
### Requirement: Отсутствие приоритета
Система SHALL NOT предоставлять пользователю выбор или изменение приоритета желания.

#### Scenario: Создание желания
- GIVEN пользователь на экране создания
- THEN отсутствует поле или переключатель приоритета
```
